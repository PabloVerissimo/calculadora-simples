A calculadora simples foi criada com o Python "Puro", sem uso de bibliotecas. A seleção de operação é feita por meio dos numeros que foram atribuidos a elas, por exemplo:

[1] Soma
[2] Multiplicação
[3] Divisão
[4] Subtração
[5] Sair

Para selecionar a operação de soma, basta que você digite o número > 1 <, e para encerrar a calculadora, basta que você digite o número 5 no campo de seleção de operações.
